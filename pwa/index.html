tok-scrape

Installable companion for the Key Metrics bookmarklet.

You're offline. Cached content is available; Sheet writes will require a connection.

1. Install the app

Tap Add to Home Screen (iOS) or Install app (Chrome/Edge) to keep this page a tap away. It works offline once installed.

checking…

2. Grab the bookmarklet

Drag the link below to your bookmarks bar (desktop) or long-press → Add bookmark (mobile). It scrapes the TikTok Shop analytics page you're viewing and (optionally) ships the result to a Google Sheet.

Log Key Metrics

Personalise the endpoint/token on the Setup tab.

3. Use it

  1. Visit partner.us.tiktokshop.com/compass/video-analysis.
  2. Select a creator and a month range.
  3. Click the Log Key Metrics bookmarklet.
  4. Open DevTools → Console to see the payload, or check your Google Sheet.

Personalise the bookmarklet

Values are stored locally in this browser only (via localStorage). Tap Generate to refresh the bookmarklet above.

Payload shape

      
{
  creator,
  scrapedAt,                 // ISO timestamp of the scrape
  dateRange: { start, end }, // selected Month date-range picker values
  metrics: [ { name, value, trend } ],
  videos:  [ { "Video ID", "Name", "Link", "Posted", "Duration",
               "Affiliate GMV", "Items sold", "Est. commissions",
               "Direct GMV", "RPM", "Views", "Completion rate", "Details" } ]
}


      
Google Sheet setup

      
The bookmarklet can POST each scrape directly into a Google Sheet via a Google Apps Script Web App.

      
    
        
  1. Create a new Google Sheet. Tabs Metrics and Videos are created on first write.
    2. 
        
  2. Open Extensions → Apps Script and paste the contents of
          apps-script/Code.gs. Save.
    3. 
        
  3. Project Settings → Script properties: add TOKEN with a random shared secret.
    4. 
        
  4. Deploy → New deployment. Type: Web app. Execute as Me, Access Anyone. Copy the /exec URL.
    5. 
        
  5. Paste the /exec URL and TOKEN on the Setup tab and tap Generate.
    6. 
      


      
Compared to the Cordova version

      
    
        
  • No native build. No config.xml, no Gradle, no Xcode — publish as static files on GitHub Pages.
    • 
        
  • Installable. iOS Add to Home Screen, Android/Desktop Install app via beforeinstallprompt.
    • 
        
  • Offline. Service worker caches the shell (HTML/CSS/JS/icon) with a cache-first strategy; Sheet writes degrade to queued retries when back online.
    • 
        
  • Auto-update. The service worker picks up new static files on next launch; no store review.
    • 
      

    


    
    

      
About

      

        tok-scrape is a tiny TikTok Shop Partner Center scraper. The scraping
        itself still runs as a bookmarklet because a PWA can't execute in the TikTok Shop origin
        — but this PWA is the installable home for the setup flow, docs, and a local-only
        endpoint editor.
